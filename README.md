# m-milan-guide

## 项目简介

`m-milan-guide` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，主要用于存放和展示一系列独立的网页内容，方便用户直接通过浏览器访问这些页面。

## 目录说明

```
m-milan-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：该目录下存放所有归档的 HTML 页面，每个页面为一个独立的文件或子目录。
- `assets/`：用于存放页面引用的资源文件，如图片、CSS、JavaScript 等。
- `index.html`：可选，作为仓库的入口页面，可提供页面导航或索引。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间没有依赖关系。每个页面可以直接通过浏览器打开，无需额外配置服务器环境。页面内容可能涉及各类主题，但均不指向任何具体网站或域名。

- 页面文件命名建议使用英文小写，单词间以连字符 `-` 分隔。
- 每个页面建议包含完整的 HTML 结构（`<html>`, `<head>`, `<body>` 等）。
- 页面中的资源引用请使用相对路径，以确保在本地或 GitHub Pages 上均可正常访问。

## 维护说明

本仓库由维护者定期归档和更新页面。如需提交新页面或修改现有页面，请遵循以下原则：

1. 所有页面应保持独立性和完整性，避免跨文件引用。
2. 页面内容应符合 GitHub 社区准则，不包含违法、违规或恶意信息。
3. 提交前请检查页面是否能正常显示，并确保资源路径正确。
4. 建议在 `pages/` 下按主题或日期创建子目录，以便管理。

## 如何使用

您可以通过以下方式访问本仓库中的页面：

- 直接克隆仓库到本地，然后用浏览器打开 `pages/` 目录下的任意 HTML 文件。
- 如果仓库启用了 GitHub Pages，可通过对应的 Pages 链接访问 `pages/` 下的页面。
